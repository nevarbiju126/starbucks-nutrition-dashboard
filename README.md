##  Key Insights
-  White Chocolate Mocha is the most calorie-packed drink (510 cal)
-  Java Chip has 84g of sugar — nearly 3x the daily limit!
-  Brewed Coffee has the highest caffeine (410mg)
-  Food items (356 cal avg) have nearly double the calories of drinks (193 cal avg)
-  Smoothies are the highest calorie drink category

##  Dataset
- `starbucks-menu-nutrition-drinks.csv` — 177 drinks
- `starbucks-menu-nutrition-food.csv` — 114 food items
- `starbucks_drinkMenu_expanded.csv` — 242 drinks with full nutrition

##  How to Run
1. Open the `.ipynb` file in Google Colab
2. Upload the 3 CSV files
3. Run all cells top to bottom
